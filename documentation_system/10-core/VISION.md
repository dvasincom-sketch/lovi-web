## Миссия
Insalon — это SaaS управленческая аналитика для владельцев beauty/SPA салонов, работающих на YCLIENTS CRM. Главная цель: предоставить прозрачный управленческий дашборд с P&L, эффективностью мастеров, обязательствами и денежным потоком.

## Контекст
Разработано для сети HeadSPA + массажного салона (headspa.beauty, Москва), с перспективой масштабирования на другие салоны.

## Границы продукта
Insalon покрывает:
- Финансовый учёт и отчётность на базе YCLIENTS-данных
- Расчёт зарплат и смен
- Управление обязательствами
- Проверки качества данных
- Интеграции (Fitmost, ЮKassa, Supabase)

Lovi — отдельный продукт (клиентская витрина для бронирования слотов).
